## 追加方法

新規集約を追加する場合:

1. 本テーブルに行を追加（コード側と docs 側の両方）
2. `docs/design/aggregates/{aggregate}/` を作成
3. `ドメイン設計.md` と `データモデル設計.md` を作成
4. `docs/design/aggregates/README.md` の集約一覧に行を追加
5. 業務知識定義書に該当機能の節があることを確認（なければ先に改訂）
6. 集約間イベントが発生する場合は [Outbox設計書](./design/cross-cutting/outbox.md) §4 にイベント追加

## 命名規則

- コード側モジュール: snake_case（例: `manage_url_delivery`）
- docs側ディレクトリ: kebab-case（例: `manage-url-delivery`）
- 集約ルート名: PascalCase（例: `ManageUrlDelivery`）
